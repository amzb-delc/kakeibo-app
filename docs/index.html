enteredBy を内部タグ配列 tags へ一般化。夫婦タグ spouse:N／カードタグ card:〇〇、明細行の識別ドット・全体カードの夫婦フィルタ（feat/expense-tags）。
全体カードの6ヶ月比較（積み上げ棒） ✅ 実装完了
全体カードを横スライド式2ペインに。単月ドーナツ ⇔ 直近6ヶ月のカテゴリ別積み上げ棒（自前SVG・色共通化・カテゴリ単独比較・タグフィルタ連動）。API に sixMonths 追加（feat/six-month-chart）。
